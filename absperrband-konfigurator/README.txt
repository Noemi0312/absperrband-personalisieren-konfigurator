1. Alle Dateien dieses Ordners in GitHub hochladen.
2. Danach in Vercel das GitHub-Repository importieren.
3. Die Systembilder liegen im Ordner public/system-images.
4. Wenn du farbige Systembilder hast, kannst du sie ebenfalls dort hochladen.

- stopper-master.jpg
- stopper-point.jpg
- stopper-twin.jpg

- tempaline-style.jpg

Optionale farbige Varianten:
- stopper-master-blau.jpg

Analog auch für die anderen Systeme.

- tempaline-blau.jpg
- tempaline-rot.jpg
